# Qaitbay Citadel

[Live Demo](https://yasmine-mustafa-web.github.io/QAITBAY/)

## Features

* **Historical Guide:** Detailed sections covering the origin of the fortress, its building year (1477 AD) and its historical significance built on the ruins of the Ancient Lighthouse.
* **Gallery:** A grid presents pictures of the castle.
* **Visitor Information:** A structured, easy-to-read section providing essential travel details including daily opening hours, ticket prices for different categories, location mapping and the best times to visit.
* Responsive for all screens.

## Tech Stack

* **HTML**
* **CSS & Bootstrap:** cards and UI.
* **JavaScript:** for responsiveness
