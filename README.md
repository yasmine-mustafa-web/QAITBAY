# Qaitbay Citadel

[Live Demo](https://yasmine-mustafa-web.github.io/QAITBAY/)

An interactive and visually engaging landing page dedicated to the historic Qaitbay Citadel in Alexandria, Egypt.

## Features

* **Complete Historical Guide:** Detailed sections covering the origin of the fortress, its building year (1477 AD) and its historical significance built on the ruins of the Ancient Lighthouse.
* **Interactive Media Gallery:** A grid layout showcasing high-quality visuals of the castle's interior, towers and defense walls.
* **Visitor Information Hub:** A structured, easy-to-read section providing essential travel details including daily opening hours, ticket prices for different categories, location mapping and the best times to visit.
* **Modern UI/UX Design:** A clean, organized, and elegant interface utilizing a refined color palette, sophisticated typography and cohesive card components.
* **Fully Responsive Layout:** Optimized for all screen sizes, from mobile devices to desktop displays

## Tech Stack

* **HTML**
* **CSS & Bootstrap:** Custom styling paired with Bootstrap components for responsive layouts, cards, grids and modern UI elements.
* **JavaScript:** for dynamic responsiveness

## Sections Included

* **Hero Section:** Features a visual of the citadel with an integrated navigation menu and a clear call-to-action to explore.
* **About/History Section:** Highlights fast facts and timelines about the fortress using a modern card-based system.
* **Gallery:** A beautiful photo grid displaying the architectural details of the site.
* **Plan Your Visit:** Provides clear informational blocks regarding ticketing and hours.
* **Footer:** Contains quick links, copyright info and simulated contact/social channels.
